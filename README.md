# DProject

## For right functionality install requirements.txt

use:
>  pip install -r requirements.txt

>  To run the file compose docker file with docker compose up

>  Run frontend by cd .\frontend\ and npm run dev

>  API is accasable via http://localhost:8000/docs

>  Frontend via http://localhost:3000/home

> To try out register, log in and basic functionality is to be observed in docker terminal and some parts in web console
